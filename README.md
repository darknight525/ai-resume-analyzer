# ResumeIQ AI 🚀

ResumeIQ AI is an AI-powered resume analysis platform that helps users improve their resumes by providing ATS scoring, skill gap detection, resume feedback, and career recommendations.

## Features

* 📊 ATS Score Analysis
* 🎯 Skill Gap Detection
* 🤖 AI-Powered Resume Feedback
* 💼 Career Recommendations
* 🎤 Interview Preparation Suggestions
* 📄 Resume Preview & Report Dashboard
* ☁️ Cloud Storage Integration using Puter

## Tech Stack

### Frontend

* React
* TypeScript
* Tailwind CSS
* React Router

### Backend & Services

* Puter.js
* AI Resume Analysis
* KV Storage
* File Storage

## How It Works

1. Upload your resume (PDF).
2. Enter company name, job title, and job description.
3. AI analyzes the resume.
4. Receive:

   * ATS Score
   * Resume Feedback
   * Skill Gap Analysis
   * Career Recommendations
5. Track all reports from the dashboard.

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/resumeiq-ai.git

cd resumeiq-ai

npm install

npm run dev
```
